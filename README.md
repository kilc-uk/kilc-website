# KILC Website

Astro 5 static site with Tailwind CSS and TypeScript. 45 pages in English and Chinese (i18n). **Live at kilc.co.uk**

## Quick Start

```bash
# Install dependencies
npm install

# Development server (http://localhost:4321)
npm run dev

# Type checking (must run after code changes)
npx astro sync
npx tsc --noEmit

# Production build
npm run build

# Preview build locally
npm run preview
```

## Key Commands

- `npm run dev` — Start local dev server
- `npm run build` — Build static site to `dist/`
- `npx astro sync` — Generate `.astro/` types (run before `tsc`)
- `npx tsc --noEmit` — Type check (no emit)

## i18n Setup

- Default route: `/en/*` (English)
- Chinese routes: `/zh/*` with alternate slugs (see `src/i18n/utils.ts`)
- Function `t(locale, 'key.path')` for translations
- Alternate hreflang links in BaseLayout for SEO

## SEO

`src/layouts/BaseLayout.astro` emits a full SEO head on every page:

- Canonical + hreflang EN/ZH alternates
- Open Graph: `og:type`, `og:site_name`, `og:image` (with width/height)
- Twitter/X Card tags
- JSON-LD Organization + LegalService schema (every page)
- RSS feed auto-discovery link

Blog detail pages additionally emit:
- `og:type = article` + `article:published_time/author/tag`
- JSON-LD `BlogPosting` schema with cover image, keywords, inLanguage

**RSS feed**: `/rss.xml` — EN blog posts, generated by `@astrojs/rss`.

## Chat API

A floating chat widget (`ChatWidget.astro`) on every page connects to a Node.js server (`chat-api/`) that provides LLM-powered guidance about KILC services. The system prompt positions KILC as an international consultancy (London, Dubai, KL) covering 7 service areas + Guardian Shield.

- **Endpoint**: `POST /api/chat` (proxied through nginx)
- **Features**: Multi-turn conversations, EN/ZH locale support, markdown link rendering (gold brand color), malformed markdown stripping
- **Deployed**: Alongside the static site via GitHub Actions → pm2 on VPS
- **Env config**: `~/kilc-website/chat-api/.env` on VPS (not in git)

## Deployment

**Automatic via GitHub Actions on push to `main`**

1. Run `npx astro sync && npx tsc && npm run build`
2. SCP `dist/` to VPS at `~/kilc-website/dist/`
3. SCP nginx config to `~/kilc-platform/nginx/conf.d/`
4. Reload nginx in Docker (`kilc-nginx`)
5. SCP `chat-api/` to VPS, install deps, pm2 restart `kilc-chat`

Requires secrets in GitHub: `HETZNER_SSH_HOST`, `HETZNER_SSH_USER`, `HETZNER_SSH_KEY`
